# Project
This project is a Task Manager Android application developed using Java in Android Studio with Groovy build scripts.
The application allows users to create, edit, delete, and sort tasks to help manage their daily activities.
Each task contains a title, description, priority level, and status.
The project contains multiple activities, UI components, and dynamic data handling.
- Link to the video: https://youtu.be/R6_cXTF8ofI

# Installation and Run the Application
For this project Android Studio is required. Recommended latest version. 
You can manually download a project folder or clone the repository. Then open the project in Android Studio, build the project after Gradle sync finishes, and run the application.

# How to Use the Application
- Create Task(Open the app -> Click Add Task -> Enter task details -> Click Save Task)
- Edit Task(Open View Tasks -> Tap a task -> Modify the information -> Click Update Task)
- Delete Task(Open View Tasks -> Long press a task -> Confirm deletion)
- Sort Tasks(Open View Tasks -> Use the Sort spinner -> Choose sorting method)
